# Contributing

## Adding or Updating a Skill

1. Update the skill's `core/` files first.
2. Update each supported adapter.
3. Update `skill.yaml` metadata and version.
4. Add or update test scenarios under `tests/`.
5. Run `python3 scripts/validate-skills.py`.

## PR Requirements

- Explain what changed and why.
- Identify affected agents.
- Include validation output.
